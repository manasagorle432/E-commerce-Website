# E-commerce-Website
This project is a basic E-commerce website developed using HTML, CSS, and JavaScript. The website allows users to view product listings, add products to a shopping cart, and perform simple user login authentication. The main goal of this project is to demonstrate basic web development concepts and how an online shopping website works.
Features:
• Product listing page
• Add to cart functionality
• Shopping cart page
• Basic user login authentication
• Simple and clean user interface
Technologies Used:
• HTML – Used to create the structure of the website
• CSS – Used to design and style the web pages
• JavaScript – Used to add functionality like cart operations and login validation
